# Данные пользователя
Копия репозитория где хранятся данные о пользвателях. <br>
Данные лежат в файле user-info.txt
## Добавлять новых пользователей строго по формату: 
user (номер) <br>
Имя: (Имя) <br>
Фамилия (Фамилия) <br>
Возраст (Возраст) 

## Пример заполнения
user 1 <br>
Имя: Иван <br>
Фамилия: Иванов <br>
Возраст: 17 <br>
